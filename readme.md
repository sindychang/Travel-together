# 旅遊行程規劃 app

多人共編的出國旅遊行程規劃工具，沿用你現有的技術架構（單一 HTML + Firebase 即時同步 + 手機優先 + PWA）。

## 功能
- **旅程**：管理多趟旅程，每趟有名稱、目的地、起訖日、當地幣別；建立時自動依日期產生天數。
- **行程**：每趟分天（Day1/Day2…），每天可排景點 / 餐廳 / 住宿 / 交通 / 其他；支援拖拉排序、時間、Google Map 連結（可導航）、地圖嵌入預覽、每項備註（營業時間、避雷 tips）；每段可填交通方式與時間，並一鍵「查 Google 路線」帶出實際時間。
- **收藏**：口袋名單（景點 / 餐廳 / 咖啡 / 購物 / 住宿），含區域、推薦度、備註、地圖連結；可一鍵加入任一天行程。
- **備忘**：大家的需求、注意事項、必帶清單（可勾選）、實用連結（入境卡網址等）。
- **工具**：群組投票決策、天氣預報（Open-Meteo，免金鑰）、匯率換算（免金鑰）、匯出／列印 PDF、離線檢視。
- **角色**：進入時選自己是誰，可隨時新增成員、切換。

## 多人共編
資料即時同步在雲端（Firebase）。把這個網頁的網址傳給旅伴，他們打開選角色即可一起編輯，所有修改即時同步。資料路徑為 `travel/`，與你的家務 app 完全分開、互不影響。

## 部署到 GitHub Pages
把本資料夾的 `index.html`、`manifest.json`、`service-worker.js` 一起放到 GitHub repo（同一層），啟用 Pages 後用網址打開即可安裝為手機 App。

## 之後可升級（需 Google Maps API 金鑰）
目前交通時間為手動填寫＋一鍵查路線。若要「自動計算各景點間交通時間／地圖直接畫路線」，需申請 Google Maps API 金鑰（可能產生費用），屆時再告訴我即可加上。
